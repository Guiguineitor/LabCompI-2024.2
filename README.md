## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/fllpo/LabCompI-2024.2.git
```

Entre no diretório do projeto

```bash
  cd LabCompI-2024.2
```

Instale as dependências

```bash
  sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

Execute

```bash
  make
```
